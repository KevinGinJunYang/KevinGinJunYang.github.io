Roadie

Fuel cost web application for NZ roadtrips.

Roadie is a roadtrip planning calculator that provides easy fuel cost approximations given their starting point, ending point and vehicle. This calculator helps provide a simple and easy way to budget for a roadtrip around New Zealand.

Roadie was created during the create camp event for summer of tech. Along with 4 others, we had 16 hours to think, plan and develop an application using the agile approach. Roadie was our idea. It uses a New Zealand offical vehicle API that allows liscence plate look up for vehicle details. This gathers information on the fuel efficency and L/KM used so that the cost to distance is correct for different vehicles.

Roadie Building Process

Understanding Roadie

PROBLEM

Why Roadie

It's always a hassle to plan a roadtrip. You have to plan food, activities and also vehicle/travel costs, This takes a long time to do! As they say: time is money.

SOLUTION

How does Roadie help

Roadie makes easy to caluclate your travel costs. It uses your cars fuel efficency to calculate how much fuel to distance is used to calculate your fuel price for the trip according to the median price of fuel at the time.

TESTING

User Feedback

During development, in its inital phases it asked for the fuel efficeny from the user, however we found that no one knew their cars fuel efficeny! so we replaced it with a number plate lookup that can find it!

IMPROVEMENT

Future improvements

We know that roadie can be imporved alot. We are currently on the search for a live api of the changing fuel prices for NZ especially if its region based.
